Aqui contarei como foi o meu Nono dia do projeto de 100 dias

Experiência

Hoje foi um dia de estudo apronfudado de CSS, confesso que CSS é algo bem complexo, então demorei a entender o que se tratava o conteúdo, entretanto, consegui sair bem sucedido, aprendi sobre o Flow do HTML e seus conteúdos no CSS.

Conteúdo

Bom, hoje aprendi muito sobre o flow do HTML, conteúdo que me ajudou a entender melhor como estruturar um website, o posicionamento etc. O que aprendi hoje foi cinco propriedades para ajustar os elementos HTML no Browser, primeiro, Position no CSS, e os atributos static, relative, absolute, fixed e sticky, estudei também sobre Z-index, que é basicamente as camadas do site. Além disso, estudei sobre Display, que é algo bem importante na estruturação do site, vi 3 atributos bastante utilizados, inline, block e inline-block. Também vi sobre o Float, que é algo bem útil para mover um elemento, e por último, o Clear.
